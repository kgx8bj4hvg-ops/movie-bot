#  Movie Recommendation Telegram Bot

##  Description
This project is a Telegram bot that recommends movies based on user preferences.

The bot interacts with the user step-by-step using buttons and suggests movies based on:
- Genre
- Production country
- Minimum rating

- The bot returns from 1 to 10 movies, sorted by the number of matches
- If no movies are found → Nothing found
- Fully interactive menu with buttons - no manual input
---

##  Features

-  Interactive movie selection (buttons)
-  Filter by production country
-  Filter by minimum rating
- 🎲 Random movie recommendation
- 🔄 Reset option
